# Chatsworth Jumping Castle Hire — Website

A modern, desktop-optimised static website for **Chatsworth Jumping Castle Hire**, built with plain HTML, CSS, and JavaScript. No frameworks or build tools required — open `index.html` directly in any browser.

---


## Sections

| Section | Description |
|---|---|
| Navbar | Fixed top bar with links: Home, Inventory, Info, Contact, Shop |
| Hero | Full-width banner with headline and CTA button |
| About | Company history (since 1999) and mission |
| Inflatables | 4-card grid showcasing product categories |
| Gallery | 3×3 photo grid with hover effects and click-to-enlarge lightbox |
| Testimonials | 3 customer reviews on dark navy background |
| Contact | Contact details, interactive Google Maps embed, and booking form |
| Footer | WhatsApp & Facebook links, copyright |

---

## How to View

1. Clone or download the repository
2. Open `chatsworth-website/index.html` in your browser — no server needed

---

## How to Update Gallery Photos

1. Add your new image to the `images/` folder
2. In `index.html`, find the `.gallery-grid` section
3. Update the `src` attribute of the relevant `<img>` tag to point to your new file
